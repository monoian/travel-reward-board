# 給同仁的公告文字

各位同仁好：

夏季競賽旅遊獎勵轉讓公告板已開放使用。

此平台僅提供「旅遊獎勵轉讓公告」使用，類似內部公告欄。

若你已自用、已私下找到買方或不轉讓，無需進入平台。

若你私下找不到買方，可至以下網址刊登公告：

```text
請填入正式網址
```

## 使用方式

### 我要刊登

1. 點「我要刊登」。
2. 輸入員工編號。
3. 系統會自動帶出姓名、單位、國家。
4. 填寫希望售價、是否可議價、聯絡方式、備註。
5. 送出後公告會顯示在列表。

### 已售出

如果旅遊獎勵已售出，請：

1. 點「管理我的公告」。
2. 輸入員工編號。
3. 點「已售出」。

## 注意事項

1. 本平台不處理付款、議價、匯款與成交確認。
2. 請買賣雙方自行聯絡與確認。
3. 公告預設 7 天後自動過期。
4. 請勿刊登不實資訊。
5. 請勿冒用他人工號。
6. 公開公告列表不會顯示員工編號。

謝謝大家配合。
