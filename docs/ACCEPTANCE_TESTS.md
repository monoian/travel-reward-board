# 正式上線驗收清單

## 首頁

| 測試項目 | 預期結果 |
|---|---|
| 打開 `/` | 可看到公告列表頁 |
| 沒有公告時 | 顯示目前沒有刊登中公告 |
| 有 published 公告時 | 公開列表顯示公告 |
| sold 公告 | 不顯示 |
| closed 公告 | 不顯示 |
| expired 公告 | 不顯示 |
| 公開列表 | 不顯示員工編號 |
| 國家篩選 | 正常篩選 |
| 價格排序 | 正常排序 |
| 手機畫面 | 可閱讀、按鈕好點 |

## 我要刊登

| 測試項目 | 預期結果 |
|---|---|
| 輸入不存在工號 | 顯示查無得獎資料 |
| 輸入存在工號 | 自動帶出姓名、單位、國家 |
| 價格留空 | 不可送出 |
| 價格輸入文字 | 不可送出 |
| 聯絡方式留空 | 不可送出 |
| 正確填寫後送出 | 直接上架 |
| 同一工號再次刊登 | 顯示已有刊登中公告 |

## 管理我的公告

| 測試項目 | 預期結果 |
|---|---|
| 輸入不存在工號 | 顯示查無得獎資料 |
| 輸入有公告的工號 | 顯示自己的公告 |
| 修改價格 | 成功更新 |
| 修改聯絡方式 | 成功更新 |
| 標記已售出 | status 變 sold，首頁不顯示 |
| 不賣了下架 | status 變 closed，首頁不顯示 |
| 一般同仁修改姓名單位國家 | 不允許 |

## 管理員

| 測試項目 | 預期結果 |
|---|---|
| 未登入進 `/admin/index.html` | 導到登入頁或顯示未授權 |
| 錯誤密碼 | 不可登入 |
| 正確密碼 | 可登入後台 |
| 後台統計 | 顯示得獎名單、刊登中、已售出、已下架、已過期數量 |
| 查看所有公告 | 正常顯示 |
| 狀態篩選 | 正常 |
| 國家篩選 | 正常 |
| 搜尋姓名 | 正常 |
| 搜尋單位 | 正常 |
| 搜尋員工編號 | 正常 |
| 管理員強制下架 | 首頁不顯示 |
| 管理員恢復公告 | status 回 published，首頁顯示 |
| 操作紀錄 | 有記錄 action、工號、IP、user_agent、時間 |

## 得獎名單匯入

| 測試項目 | 預期結果 |
|---|---|
| 英文標題 CSV | 可匯入 |
| 中文標題 CSV | 可匯入 |
| 從 Excel 複製貼上 | 可匯入 |
| 重複員工編號 | 更新既有資料 |
| 缺少員工編號 | 顯示錯誤 |
| 缺少姓名 | 顯示錯誤 |
| 缺少單位 | 顯示錯誤 |
| 缺少國家 | 顯示錯誤 |

## 自動過期

| 測試項目 | 預期結果 |
|---|---|
| 公告未過期 | 首頁顯示 |
| 公告超過 expires_at | 首頁不顯示 |
| 過期公告 | 後台看得到 |
| API 查詢後 | 可將過期 published 更新為 expired |

## 安全與防呆

| 測試項目 | 預期結果 |
|---|---|
| GitHub 沒有真正管理員密碼 | `.env.example` 只有範例 |
| 公開列表不顯示員工編號 | 通過 |
| API 不回傳不必要敏感欄位 | 通過 |
| 管理員 cookie 使用 httpOnly | 通過 |
| 一般同仁只可改自己的公告 | 通過 |
