# 管理員操作手冊

## 管理員入口

```text
/admin/login.html
```

輸入 Cloudflare 環境變數設定的管理員帳號密碼。

## 第一次啟用

1. 登入管理員。
2. 進入 `/admin/setup.html`。
3. 點「初始化資料庫」。
4. 進入 `/admin/winners.html`。
5. 匯入得獎名單。
6. 回首頁測試刊登。

## 匯入得獎名單

支援中文欄位：

```csv
員工編號,姓名,單位,國家
A001,王小明,業務一處,日本
A002,李小華,北區單位,韓國
```

也支援英文欄位：

```csv
employee_no,name,unit,country
A001,王小明,業務一處,日本
A002,李小華,北區單位,韓國
```

## 查看公告

在 `/admin/index.html` 可查看：

- 刊登中
- 已售出
- 已下架
- 已過期

可以依國家、狀態、姓名、單位、員工編號搜尋。

## 處理錯誤公告

如果公告內容不合規，可以：

1. 找到該公告。
2. 點「下架」。
3. 公告會從公開列表移除。

## 恢復誤下架公告

如果公告被誤下架，可以在後台找到該公告後點「恢復」。

## 設定公告有效天數

預設 7 天，可在管理員後台設定，例如 3 天、5 天、7 天或 14 天。

## 操作紀錄

系統會記錄：

- 使用的員工編號
- 操作類型
- 公告 ID
- IP
- 瀏覽器資訊
- 操作時間

因為一般同仁只用工號操作，管理員應定期查看是否有異常。
